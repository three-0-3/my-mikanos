# my-mikanos
Shadowing implementation of the educational operating system 'mikanos'

## Files

- MikanLoaderPkg
    - The MikanOS loader as a UEFI Application
- kernel
    - The MikanOS kernel
- resource/ipaexg.ttf
    - IPA gothic font file
- IPA_Font_License_Agreement_v1.0.txt
    - License agreement for IPA fonts
